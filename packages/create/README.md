# `feup-create`

> 查看详情： https://www.feup.cn
> 服务于 `feup-cli`, 提供创建项目的功能

## Usage

```js
npm install feup-create
// 或
yarn add feup-create

require("feup-create")(name, options);
```

| 参数名    | 说明                            |
| --------- | ------------------------------- |
| `name`    | 要创建的项目名称                |
| `options` | commander.action 提供的 options |
